# BottleFind 文档导航

该目录提供 BottleFind 的产品说明、接入说明与本地开发指南。

## 文档列表

- `LOCAL_SETUP.md`：本地部署与启动步骤
- `API_INTEGRATION.md`：外部数据源与接口集成说明
- `REQUIREMENTS.md`：需求与功能清单

## 推荐阅读顺序

1. 先阅读 `LOCAL_SETUP.md` 完成本地运行。
2. 再阅读 `REQUIREMENTS.md` 理解产品边界。
3. 最后查看 `API_INTEGRATION.md` 做二次扩展。

bottlefind-app/
